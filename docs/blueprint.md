# **App Name**: NutriTrack

## Core Features:

- Daily Food Logging: Allow users to input consumed food items and their quantities. The system will automatically calculate associated caloric and nutrient values based on predefined food data.
- Pre-defined Food Nutrition Customization: Allow users to modify pre-defined food items' nutritional information if required.
- Daily Calorie Summary: Display a clear overview of total daily calorie and macronutrient intake, along with a detailed nutrient breakdown.
- AI Weekly Report Generation: An AI tool to generate personalized weekly reports, summarizing intake patterns, identifying trends, and offering encouraging, basic dietary suggestions based on logged data.
- Automated Email Delivery: Automatically send the AI-generated weekly calorie and nutrient report to the user's registered email address.

## Style Guidelines:

- Color Palette based on health, clarity, and personal progress: A light color scheme for a clean and refreshing feel.
- Primary color: A vibrant, clear blue (#1481B8) representing focus and vitality. (HSL: 200, 80%, 40%)
- Background color: A very light, desaturated blue (#F0F3F5) providing a clean canvas. (HSL: 200, 20%, 95%)
- Accent color: A supportive purplish-blue (#5366C6) for highlighting key information or actions. (HSL: 230, 50%, 55%)
- Headline and Body font: 'Inter', a grotesque-style sans-serif for its modern, neutral, and highly readable qualities across various screen sizes.
- Use clean, minimalist line icons related to food, health, and tracking, maintaining a modern and approachable aesthetic.
- Implement a clean, card-based layout with a strong emphasis on mobile responsiveness for easy access and data entry on a phone, as requested.
- Incorporate subtle, fluid animations for feedback during food logging, data updates, and report generation, enhancing user interaction without distraction.
